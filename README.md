# ice-cream-shop
This assignment represents a basic Java input/output and variable program using a pretend ice cream shop order system.
It teaches these programming concepts:
Scanner Objects → Used to get input from the user.
Variables → Storing information like flavor, scoops, and total price.
User Input → Asking the user questions and saving their answers.
Math Operations → Calculating the total cost.
Output Statements → Printing an order summary.
